RAWG is the largest video game database and game discovery service. As a company that believes in sharing and openness, RAWG offers a comprehensive API to access over 500,000 games, along with detailed data, search functionalities, and machine learning recommendations. This API provides an unparalleled resource for developers, researchers, and gaming enthusiasts to build innovative applications and gain insights into the gaming world.

Key Features

  • Extensive Database: Access a vast library of games, including detailed information such as game titles, release dates, genres, platforms, developer details, and more.
  • Search and Filtering: Utilize powerful search and filtering capabilities to find games based on various criteria, such as genre, platform, release date, and popularity.
  • Game Recommendations: Leverage machine learning algorithms to receive personalized game recommendations based on user preferences and behavior.
  • Media and Visuals: Access screenshots, videos, and other media assets for games to enhance your application or website.
  • User Ratings and Reviews: Integrate user ratings and reviews to provide insights and feedback on games from the gaming community.

Use Cases

The RAWG API can be used in various applications, including:

  • Game Discovery Platforms: Create platforms that help users discover new games based on their interests and preferences.
  • Game Analytics and Research: Use the extensive game data for research, market analysis, and trend tracking in the gaming industry.
  • Game Reviews and Ratings: Build websites or apps that feature game reviews, ratings, and user feedback to guide players in choosing games.
  • Game Collection Management: Develop tools for managing and displaying personal game collections, with detailed information and media.
