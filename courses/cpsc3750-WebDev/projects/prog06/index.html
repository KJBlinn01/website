Purpose Statement

The purpose of this assignment is to provide students with hands-on experience in creating interactive web elements using HTML, CSS, and JavaScript. By developing a dynamic webpage that allows users to generate colored buttons, change button colors, calculate running totals, and implement button movement with timer events, students will enhance their understanding of event handling and manipulation in web application development.

Description

This assignment focuses on enhancing students' proficiency in web development by creating an interactive webpage that involves generating colored buttons, tracking running totals, and implementing button movement with timer events. Students will apply their knowledge of HTML, CSS, and JavaScript to craft a dynamic user interface that responds to user actions and provides engaging functionality. By completing this assignment, students will strengthen their skills in event handling, element manipulation, and user interaction in web application development.

Instructions

  1. Create a webpage named `ButtonMove.html` along with associated CSS and JavaScript files.
  2. Upload the files to your hosted website for accessibility.
  3. Implement the following features on the webpage:
    • Include a list and a button at the top of the page.
    • Create a viewing area (rectangle) that occupies approximately 3/4 of the browser window, colored black.
    • Develop a dropdown list of colors (Red, Green, Blue, Yellow, White).
    • Add a button labeled "Make Button" that generates a colored button of the selected color at a random location within the viewing area with a label of a random integer.
    • Ensure that the generated button stays completely within the viewing area to adjust random locations accordingly.
    • When a colored button is clicked, change its color to the currently selected color from the color list.
  4. **Grade C Features:**
      Include the functionality mentioned above.
  5. **Grade B Features (in addition to Grade C):**
    • Display a running total of the sum of all clicked buttons using the random number as the button text. Place this total with an appropriate label near the other controls.
  6. **Grade A Features (in addition to Grade B):**
    • Implement timer events to make the buttons move around the viewing area in random directions.
    • Buttons should change direction upon reaching the edge of the viewing area.
    • Add a button labeled "MOVE" at the top to initiate button movement for all buttons. Change the button text to "PAUSE" to halt movement when pressed again.
  7. Ensure the inclusion of a navigation bar for easy access to different sections of the webpage.
