Project 2: Collect App Phase 1

For this project you will build phase 1 of a collection web application. An app that allows the user to search through an extensive collection of something (of your choosing), display stats about the collection, display search results, etc. As you have learned several required aspects of web development, with AJAX, you now have enough knowledge to begin planning your next project. You will build a web app to support collections. For example, I collect baseball trading cards. Think about something you enjoy collecting or would like to collect. Find an API source where you can get information about it. Build an app to allow you to build your collection. Build an app that allows you to search for them, in multiple ways and display them in a list format as well as drill down to a detailed, single item, format.
